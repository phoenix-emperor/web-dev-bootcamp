Your 4-Week Journey to Web Development

Week 1

HTML Basics & Page Structure

Theme: "Let's Build Our First Web Page"
  • Day 1: HTML boilerplate, headings, paragraphs, lists, links
  • Day 2: Images, tables, forms
  • Day 3: Semantic tags, page structure
Project: Start building your personal portfolio

Week 2

CSS Basics & Styling

Theme: "Making It Beautiful"
  • Day 4: CSS syntax, colors, fonts
  • Day 5: Box Model, margins, padding
  • Day 6: Flexbox for layout
Project: Style your portfolio with CSS

Week 3

Advanced CSS

Theme: "Modern & Responsive Design"
  • Day 7: Media Queries
  • Day 8: Pseudo-classes, transitions
  • Day 9: Google Fonts, animations
Project: Make your portfolio responsive

Week 4

JavaScript & Deployment

Theme: "Bringing It to Life"
  • Day 10: JS basics, variables, functions

  • Day 12: GitHub Pages deployment
Project: Deploy your portfolio
